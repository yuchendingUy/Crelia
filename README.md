## 项目简介

Crelia 基于 [Folia](https://github.com/PaperMC/Folia) 开发。

初衷是为了使机械动力服务器利用folia的多核心方案以缓解卡顿。
## 当前状态

- 以folia作为基础叠加neoforge功能，理论上性能十分接近folia。理论上！
- 适配中，目前大多模组都会与其不兼容
